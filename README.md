# simple-auth
Simple JWT Auth Application, SpringBoot backend and NextJS frontend, Docker as deployment environment.

## build
- [OpenJDK 16 OpenJ9](https://adoptopenjdk.net/)
- [Maven](https://maven.apache.org/download.cgi)
- [Docker](https://docs.docker.com/get-docker/)

```
mvn clean install
```
