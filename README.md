# simple-auth
Simple Auth Application, SpringBoot backend and Angular frontend, Docker as deployment environment.

## build
- [Eclipse OpenJ9 VM AdoptOpenJDK](https://adoptopenjdk.net/)
- [Maven](https://maven.apache.org/download.cgi)
- [Docker](https://docs.docker.com/get-docker/)

```
mvn clean install -Pdocker
```
